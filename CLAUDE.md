# Claude Code 전용 프로젝트 규칙

## 절대 규칙
- 기존 파일 구조를 임의로 파괴하거나 대규모 리네이밍하지 않는다.
- 인증 세션 상태는 반드시 `Zustand`의 `useAuthStore`를 사용한다.
- 스타일은 `Tailwind CSS` 토큰(`bg-brand`, `text-muted`, `border-border`)을 우선 사용한다.
- 인증은 `Supabase Auth`를 기준으로 구현하며, 한국형 소셜 로그인(카카오/네이버) 흐름을 유지한다.
- 결제는 `토스페이먼츠 위젯` 기반으로 구현하고 성공/실패 콜백 라우트를 유지한다.

## 개발 원칙
- 코드 변경 시 비개발자도 이해할 수 있는 명확한 이름을 사용한다.
- 복잡한 로직은 짧은 주석으로 의도를 설명한다.
- 신규 기능 추가 시 랜딩/결제/법적 문서의 연결 동선을 깨지 않는다.
